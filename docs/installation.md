# Installation

deezer-ts is available as an npm package. You can install it using npm, yarn, or pnpm.

## Using npm

```bash
npm install deezer-ts
```

## Using yarn

```bash
yarn add deezer-ts
```

## Using pnpm

```bash
pnpm add deezer-ts
```

## Requirements

- Node.js 16 or later
- TypeScript 5.0 or later (if using TypeScript)
